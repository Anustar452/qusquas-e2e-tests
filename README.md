# Qusaqus Automated E2E Testing

This project contains automated end-to-end tests for Qusaqus using Playwright and Node.js.

## Test Types
- Static UI & accessibility tests
- Dynamic user-flow tests (login, posting ads, core features)

## Tools
- Node.js
- Playwright
- JavaScript

## How to Run
npm install  
npx playwright install  
npx playwright test
